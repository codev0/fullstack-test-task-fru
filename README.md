# Тестовое на вакансию Fullstack Node.js Developer

Чтобы посмотреть как работает запустите `pnpm demo`.

Работает через сниппет тк это более надежное решение:
1. не блокирует поток отрисовки тк трекер подгружается асинхронно 
2. собирает события раньше насколько это возможно
3. после загрузки (IIFE) и инциации трекера все события накопленные во временной очереди будут обработаны

## Структура папок
```
public для раздачи статики трекера
demo для проверки трекера в браузере через пакет serve
src серверный код
tracker код трекера который билдится в паблик и раздается сервером из папки public
```

## Код бэкенда
```
├── src
│   ├── plugins // настроенные плагины фастифай для автолоудера.
│   ├── routes // контроллеры.
│   ├── schemas // схемы данных.
│   ├── app.ts // инициализируем плагины, роуты, и переопределяем дефолты.
│   └── server.ts // инициируем апку, подбрасываем в сервер, запускаем сервер.
```
